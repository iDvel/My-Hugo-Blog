Python 学习计划

2018-08-27 | 阅读

文章目录

只是为了设定计划与监督自己，记录下自己的学习路径。

《笨办法学Python 3》（Learn python the hard way）

读了几章发现学习模式过于基础，比较适合第一次编程的纯新手。

《Python 学习手册》（Learning Python）

网上大部分评论为：啰嗦 / 细致，可能和封面一样，喜欢细嚼慢咽的朋友可以慢慢的啃这本书。。。

《Python编程：从入门到实践》（Python Crash Course）

如书名，看目录能看出来，第 12 章后开始是各种小项目，甚至还有 Django 的小入门教程。

最终还是选择这本书入门。

前 7 章是中规中矩的语言入门，翻译质量很好。

第 8 章开始涉及的门类较多，基本上都属于程序员必备技能，都是点到为止，看起来也不困难，可以增加知识面。

学习时长：69 个番茄

PythonCaff 社区，基于 3.7 的最新翻译版，官方的入门教程，过一遍准没错。

感觉第 8、9 章的翻译有点生涩难懂。

稍微有点收获吧，比如上本书没有讲清楚 Python 中各种 import 语句的不同效果。

学习时长：19 个番茄

我觉得阅读完一门基础书籍后再看廖老师的教程比较合适，廖老师的教程精简干练，适合查漏补缺。

学习时长：50个番茄

又重新过了一遍，最后的实战没看，感觉没有承前启后看的非常费解，准备直接看看其他 web 的书籍了。
