新 MBP 到手后，我就预感要折腾好一阵子，因为之前把系统整的太凌乱，决定不使用 Time Machine，从头开始对 MBP 新装新居，每做几步就做个简单的记录，写着玩玩吧，好久都不写博客了。

上手体验

新 MBP（2018 15寸） 对比我的老款 MBP（2014 mid 15寸）

  • 信仰灯几年前就开始不亮了。。。
  • bar 上的 esc 键可能需要太长时间来适应，居然还不在最左边！
  • 蝶式键盘的键程极短，基本已经适应
  • 方向键的 ↑ ↓ 感觉比老款增大了误触几率
  • 默认分辨率稍高（1680 x 1050），老版为（1440 x 900）
  • 原彩显示，体验极佳！
  • 触摸板过大，但做了很好的防误触处理
  • 指纹识别，not bad
  • USB-C * 4 。。。
  • 发烫，通病，4 年了还是老味道
  • 独显性能提升（毕竟 4 年了，老 Mac 玩影子战术低档还卡，新 Mac 高档流畅）

配件

  • Apple USB-C 转 USB 转换器
  • 戴尔 DA300

偏好设置

  • 系统所有偏好设置

    • 通用：关闭文稿时要求保存更改
    • Dock：将窗口最小化为应用程序图标，连按最小化
    • 吮吸效果：defaults write com.apple.dock mineffect suck; Killall Dock
    • 添加空白格：defaults write com.apple.dock persistent-apps -array-add '{"tile-type"="spacer-tile";}'; Killall Dock
    • 堆栈区网格高亮：defaults write com.apple.dock mouse-over-hilite-stack -bool TRUE;killall Dock
    • 调度中心：关闭仪表盘，设置触发角（右下角显示桌面）
    • 触摸板：辅助点按、轻点、滚动方向
    • Siri：关闭
    • 辅助功能 - 鼠标与触控板 - 触控板选项：启动拖移(三指拖移)

  • Finder 偏好设置

    • 边栏调整
    • 显示所有文件扩展名
    • 显示 - 显示各种栏
    • 标题栏显示完整路径：defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES;killall Finder

  • 部分快捷键重设（极具个人风格）

    • 双击 command：Alfred
    • command + 空格：切换输入法
    • option + 空格：Alfred - File search - Navigation
    • control + 空格：spotlight
    • shift + 空格：iTerm

  • 禁用大部分 iCloud 同步

先安装能快速还原使用习惯的App

提前挂好 Xcode，下载慢

  • 输入法
  • Firefox / Chrome
  • Surge
  • Dropbox
  • Alfred
  • 1Password
  • Spark

至此，基本上只差功能性App了，大部分的使用习惯都找了回来

App库

开始一个一个的下载App，这时才发现还是 Time Machine 好啊。

特别注意个别 App 的反激活与同步。

  • 1Password ，妈的 7 不带老用户玩了，需订阅或再次买断
    • 之前买断的可以从 MAS 购买记录里重新下载
  • Fantastical 2 根本没有反激活。。。
  • NTFS 需要在官网反激活一下，看到 Licences 有点变化
    • Microsoft NTFS for Mac by Paragon Software 新版可用的
    • Paragon NTFS For Mac 14 (Multilingual) 好像是曾经的叫法，已不可用，下载14也无法打开

参考老Mac导出的App列表，由于数量庞大几乎大部分时间都花在App的下载与设置上了。
