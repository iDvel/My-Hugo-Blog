再见，双点医院！

终于肝完了主题医院的精神续作——双点医院。。。

太肝了。。。

这一代的病人依然没有摒弃「请病人不要死在走廊上」的优良传统，如果你把一个病人治死了，他会坚持换完衣服走出病房，然后再在走廊死掉，再化为鬼魂，为周围的病人启动一个「加速行走」的Buff。。。

因为这个加速行走的Buff，我甚至都不招聘有抓鬼技能的员工了。

游戏进程

  1. 入场
  2. 接待
  3. 诊断
  4. 治疗
  5. 离场

中途可能包含「愤怒离开」「治疗效果不佳」「死亡」。

在接受治疗前就死亡，是因为排队排太久，健康值降为 0。

愤怒离开的主要原因是心情降为 0。

诊断流程 （妈的智障）

→ 全科 → 诊断 → 全科 → 诊断 → 全科 → 治疗

如果第一次全科就100%，直接治疗。

即便到诊断室已经100%了，病人也会再次返回全科。（大夫可能是说你去照个X光回来给我看看）

如果已有的诊断室已经用完，仍没到100%，会发出消息，问你是送回家、等待、治疗？

在后期基本只有这两种情况：

  • 进场 → 全科 → 治疗 → 离场或死亡

  • 进场 → 全科 → 某个诊断室 → 全科 → 治疗 → 离场或死亡

全科技能跟上的话，很少有第三次复诊。

上表【 】中为具有诊断功能的科室。

加粗项「病房」「精神病」「DNA」都是既可以诊断又可以治疗的。

  • 培训室：一个就够，至少弄 50 个 4% 加成的书柜，增加培训效率。
  • 研究：一个都不要，之前地图的研究是通用的。（你在这关如果有新研究，回老地图研发也是可以的）

房间等级，5 级封顶，豪华房间对员工有加成，￥300 的奖章是最高性价比。

人物需求

人生在世，无外乎吃喝玩乐。

  • 温度
  • 卫生

上面两个洗手液和暖气/空调可以简单解决。

  • 吃
  • 喝
  • 玩

售货机解决吃喝。

游戏机解决玩。

咖啡厅可以同时解决「吃」和「喝」，并赐予一个开心 Buff。

  • 厕所

如果你建了一些吃喝建筑，但没有厕所。。。

他就会把新鲜的大便拉到地上，地上时而出现的一滩水就是病人撒的尿。

技能选择

简单来说，方便管理的话。。。

因为后期全科技能很高，一个往返基本能就诊断 100%。

  • 勤杂工：留一个升级，剩下全维护。
  • 客服：无脑接待
  • 护士：全部治疗，不要诊断，无需任何管理操作，自动轮换。
  • 医生：全部治疗，不要诊断，无需任何管理操作，自动轮换。

这么玩主要还是因为放弃了很多诊断室，治疗技能是为了保障治愈率。

其中需要规划一下全科大夫，全部加全科，只负责全科科室，20级以下的医院十个足够。

核心战略

  • 让员工开心，Buff 加成，咖啡厅 + 休息室 + 厕所 + 5级全科科室。（咖啡厅与厕所设置成只允许员工使用）

科室里不放咖啡机和零食，让员工休息时去咖啡厅吃喝。

  • 让病人迅速滚蛋，不要让他吃、喝、玩乐、拉撒，150天内要么把他治好，要么把他治死。

这是一个塔防游戏。

  • 让病人保持高效率，除了已经避免的吃喝和上厕所，长椅也不摆，让病人在排队时一直排排站好。

  • 无论病人得了什么病，任何诊断室（包括全科）都能检查出来

  • 地域分配——集中诊断：集中全科与诊断室，让病人走最少的步数完成诊断。

  • 治疗区：随便放，放远处，这是病人的最后一步，在进入治疗室付款前别死了就行。

  • 舍弃无用的诊断室，「病房」、「精神」、「DNA」充当诊断室。

为保持循环效率，「病房」的床位、「精神」与「DNA」的数量要适当增加。

超级磁力共振是收费最高的诊断科室，可以考虑。

前期迅速赚钱战略

落地：一接待一全科一病房。

除了病房这样的，全部按最小空间建造，方便后期调整位置。

全科慢慢增加。

病房吞吐量大，多弄几个护士和床位。

然后一个一个的补治疗室，先不补「精神」和「DNA」。

病人在完成「全科 → 病房 → 全科」这个诊断流程后就会弹出信息，诊断率只有xxx，治疗还是送回家？送去治疗就好，前期不要在乎治愈率，能赚钱就好。

期间招聘跟上，然后完善科室就可以了，这样是稳赚不赔的。

在满足人员配置后，一次性补足「精神」和「DNA」，如果先补这俩但是数量不足，病人就会去排大长队。

所有规划完毕后，上培训，慢慢就进入后期了。

摩天塔通关配置

中间为集中诊断区，左下为治疗区，右下就放了个培训室。

这是我第三次干摩天塔了，之前铺的满地图都是，13 个场地铺满 9 个了，然后铺烦了删档重来了。

这个低配简配，本来是想着舍弃外围 8 个场地，只玩内围 4 个场地的，没想到直接干到通关了。

全科：8 个房间，10 个员工。

精神：4 个房间，4 个员工。

DNA：4 个房间，4 个员工。

病房：1 个 6 床及以上的房间，4 个及以上的员工，因为员工老瞎溜达，效率不够，人数来凑。

每个治疗室 1 个，员工自己可以补三四个轮休的。

手术室太费时间，适当增加。

等级压到 20 级以下，这样全科不会产生排队。

评价

差评。

  • 没有「房间模板」或「复制房间」这个功能。（关键点，有这个功能就是好评）

前期造全科就够烦的了。

最后一关摩天塔，我第一次想铺开玩大医院，病人数量增加的太多，没有复制功能烦的我重新开档了。

  • 价格偏高，关卡游戏性重复。

  • 伤肝。。。

  • 没有一键「让所有已诊断病人滚去治疗」选项，前期拥堵的时候得一个一个点。

  • 人物 AI 智障

比如病房排队人多的时候，有人同时开门和关门，能把人挤出去。

休息室、培训室这种大量进出，员工都跟傻子似的，开关门效率低。

病房的屏风，如果你在门左右各方一个，病人只会去右边的，即便左边是空的，病人还会去右边排队，除非右边已经有人排队，他才会去左边的，利用效率低下。

排队，科室没有病人，2 号位已经站在门前，1 号位刚拉完屎从远方赶来，2 号位就干等着他。

如果你仔细看一会病房，包括骨科，护士们的治疗效率真是太低了急死人，各种不知所谓乱走动。

  • 各种 BUG，据说病房 BUG 已经修复，这个 BUG 是真tm烦。
  • 逼死强迫症！

最后一关了，45 星全部到手了，玩家要真真正正的开始创造大医院了，不能复制房间我都忍了。

结果最后一关居然是不对称的。。。最外围有两处和别的长得不一样，偏小还少了个门。

  • 管理方式弱智

玩过 Rimworld 后很难接受这种糟糕的管理方式了，尤其是最后一关铺大医院的时候，我需要统计全科的员工人数、精神和DNA的员工人数，病房专属的员工人数等等来方便我招聘，你只能一页一页的翻。
